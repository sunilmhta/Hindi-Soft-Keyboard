# Hindi Keyboard For Androids
This is a Hindi Soft Keyboard for Android Devices.
### Features
  - Hindi Alphabets 
  - Hindi Numerals
  - English Numerals
  - Rupee Symbol
  - Other Special Symbols
  - Pop up when a key is tapped
  - Pop up sound when a key is tapped
